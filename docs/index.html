The Problem

Five "solutions." None of them work for dashboards.

This Salesforce idea has been open since 2014. The ecosystem has tried — and failed — to fill the gap five different ways.

✕

Native "Subscribe"

Emails a static PNG. No branding. No multi-dashboard. No on-demand. No PDF.

✕

Native "Export"

Exports the underlying data, not the visual. You get a CSV, not a report.

✕

html2canvas hacks

Blocked by Lightning Web Security, shadow DOM, and iframe boundaries. Doesn't work.

✕

Visualforce renderAs="pdf"

Slow, ugly, no Lightning charts. The 2014 solution to a 2026 problem.

✕

Conga / Titan / S-Docs

Built for transactional documents, not dashboards. $35–$80/user/month plus per-doc fees. Data leaves your org.

✓

Crafter PDF

Native LWC. Reports API → Chart.js → jsPDF. All client-side. Your data never leaves your org.
